# AccOffice Development Guide

## Project

AccOffice is a professional accounting office management system.

## Tech Stack

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS v4
- shadcn/ui
- Prisma ORM
- SQLite
- React Hook Form
- Zod

## Coding Rules

- Never create duplicate components.
- Always reuse existing components.
- Keep folder structure clean.
- Use TypeScript everywhere.
- Use Prisma for database access.
- Use shadcn/ui for all UI.
- Prefer server components unless client components are required.
- Write clean, production-ready code.

## Main Modules

1. Dashboard
2. Customers
3. Monthly Workflow
4. Documents
5. Finance
6. Reports
7. Employees
8. Settings

## Goal

Build a production-ready accounting office management system.
