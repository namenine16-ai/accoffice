# Roadmap

## Core phases

### Phase 1 — Foundation
- Establish the Next.js app shell with App Router.
- Build the customer module with CRUD, validation, and data persistence.
- Create shared UI components and a consistent design system.

### Phase 2 — Operational insights
- Add authentication and user flows.
- Build the dashboard experience with high-level summary cards.
- Create a workflow overview for monthly accounting tasks.

### Phase 3 — Monthly workflow
- Implement workflow filters for month, year, status, customer, employee, and search.
- Add progress bars, status badges, and task detail handling.
- Support task assignment, deadlines, priorities, and task completion tracking.

### Phase 4 — Quality and polish
- Add loading skeletons, empty states, and error handling.
- Standardize spacing, typography, and responsive layout.
- Improve code quality, remove unused logic, and enforce strict TypeScript.
- Add documentation and CI validation.

## Near-term goals

- Expand reporting and finance dashboards.
- Add document management and tax filing details.
- Improve access control and employee role handling.
- Add tests and automation for regression coverage.
