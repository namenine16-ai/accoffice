# Sprint 01

## Objective

Deliver a solid customer management module as the first core feature of AccOffice.

## Completed work

- Customer list page with searchable and paginated results.
- Customer detail and edit pages with reusable form components.
- Create and delete customer workflows.
- Input validation using Zod and React Hook Form.
- Prisma-backed SQLite persistence and API endpoints for customer operations.

## Acceptance criteria

- `npm run build` passes.
- No TypeScript errors.
- Responsive UI using Tailwind and shadcn/ui.
- Reusable UI primitives for forms and tables.

## Status

Completed.
